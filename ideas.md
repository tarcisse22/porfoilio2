# Design Brainstorm — Tarcisse Ndjibu Developer Portfolio

## Selected Approach: Tech-Forward Minimalist with Light Theme

**Design Movement:** Contemporary tech minimalism with Swiss design principles

**Core Principles:**
- Clean, spacious layout emphasizing code and craft over decoration
- Subtle depth through layered cards and soft shadows
- Typography-driven hierarchy with intentional whitespace
- Tech-forward but approachable — not cold or corporate

**Color Philosophy:**
- Background: clean white (#ffffff) with subtle cool undertone
- Primary text: deep charcoal (#1a1a1a) for readability
- Accent: vibrant tech blue (#0066ff) for CTAs and highlights
- Secondary accent: soft slate (#64748b) for metadata and secondary text
- Card backgrounds: off-white (#f8fafc) for subtle differentiation

**Layout Paradigm:**
- Asymmetric hero with large display type and subtle animated gradient accent
- Project cards in a responsive grid (2-3 columns) with hover effects
- About section: two-column layout with image and text
- Contact form: centered, minimal, with inline validation feedback

**Signature Elements:**
- Thin accent line (tech blue) under section headings
- Code snippet snippets as decorative elements in background
- Subtle grid pattern in hero background
- Tech icons for skills (React, Node.js, etc.)

**Interaction Philosophy:**
- Project cards lift on hover with enhanced shadow
- Links have animated underline on hover
- Smooth scroll behavior with staggered entrance animations
- Form inputs have focus states with blue accent

**Animation:**
- Hero text staggered entrance on page load
- Project cards fade-in with slight upward drift on scroll
- Hover effects: card shadow deepens, text color shifts
- Loading states with subtle pulse animation

**Typography System:**
- Display: Inter (bold, 700+ weight) for headings
- Subheadings: Inter (600 weight, slightly spaced)
- Body: Inter (400-500 weight, generous line-height for readability)
- Code/metadata: JetBrains Mono (monospace for technical elements)
